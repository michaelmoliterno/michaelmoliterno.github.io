Download a few mta turnstile data files from http://web.mta.info/developers/turnstile.html.

You can use wget on your terminal to grab a file and place it in the current directory:

wget "http://web.mta.info/developers/data/nyct/turnstile/turnstile_150110.txt"

The process_mta_data() function is similar to get_MTA_data_by_turnstile(), but it calculates the counts on entrances/exits foe each 4 hour interval.

(It also supports a list of files, so let's load in three weeks worth of data)

In [4]:
mta_time_series = process_mta_data(['turnstile_141025.txt','turnstile_141101.txt','turnstile_141108.txt'])

Now let's get total counts by turnstile by day (measured by the total number of passengers that entered through the turnstile on each day).

The function also includes two lists that summarized data that was cleasned and sometimes modified. We will take a look at the lists to see if there are serious problems with the data.

See the details of process_mta_time_series() for more details.

# This is a dict of stations with key = station and val = % share of volume
ridership_share = {}
total_passengers = sum(total_ridership_counts)

for station_line, raw_count in total_volume.iteritems():
    ridership_share[station_line] = raw_count/float(total_passengers)
    
    
# this will similar to a CDF for stations; keys are station names, values are cumulative probabiliy of stations that 
# have greater than or equal volume

station_cdf = {}

for index, key in enumerate(sorted(ridership_share.items(), key=operator.itemgetter(1), reverse=True)):
    if index != 0:
        station_cdf[index+1] = [key[0],key[1]+station_cdf[index][1]]
        #print index
    else:
        station_cdf[index+1] = [key[0],key[1]]
      
cdf_plot = []

for x, y in sorted(station_cdf.items(), key=operator.itemgetter(0), reverse=False):
    cdf_plot.append([x, y[1]])

plt.axis([0, 460, 0, 1.02])
plt.xlabel('MTA Stations (descending by weekly volume)')
plt.ylabel('Cumulative Passenger Volume')
plt.plot(*zip(*cdf_plot))
