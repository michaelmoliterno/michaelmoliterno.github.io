Introduction

NYC Open Data is a very cool initiative that makes the wealth of public data generated by various New York City agencies/organizations available for public use. One of these data sets is MTA Turnstile data, which details entrances and exits for every turnstile in the NYC Subway system. For this project, I ingested the raw turnstile data text files, cleaned/organized the data into Python dictionaries, then performed exporatory analysis on a few weeks worth of turnstile data.

See here for an IPyton Notebook that details the analysis discussed below.
